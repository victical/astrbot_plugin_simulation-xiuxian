# AstrBot 模拟修仙插件

## 插件简介
本插件是基于 AstrBot 平台开发的模拟修仙插件，旨在为用户提供一个沉浸式的修仙体验。插件将利用 AstrBot 配置的大模型服务提供商，生成丰富的剧情内容，并结合固定的奖励机制，让玩家在修仙世界中不断成长，最终飞升成仙。

## 核心功能与设定

### 1. 游戏开始
- **初始身份**: 玩家开始游戏时，默认为一介“凡人”。

### 2. 修仙之路选择
- **门派或散修**: 玩家可以选择加入某个修仙门派，或选择成为一名自由自在的散修。
    - **门派**: 加入门派可获得门派功法、资源支持，并可能触发门派特有的剧情任务。
    - **散修**: 散修则需自行探索、寻找机缘，自由度更高，但风险也更大。

### 3. 剧情系统
- **大模型生成剧情**: 插件将利用 AstrBot 配置的大模型服务，根据玩家的选择和当前状态，动态生成各种修仙剧情。这些剧情将极大地丰富游戏体验。
- **固定奖励**: 尽管剧情内容由大模型完善，但通过剧情获得的奖励（如经验、法宝、丹药、功法等）将是预设好的，以确保游戏平衡性和可玩性。

### 4. 修仙等级体系
玩家的主线目标是提升修仙等级，逐步迈向仙人境界。等级体系如下：
- **凡人**: 修仙之始，尚未踏入修行门槛。
- **炼气**: 初窥门径，开始吸收天地灵气。
- **筑基**: 奠定道基，实力显著提升。
- **金丹**: 凝结金丹，寿元大增，神通初显。
- **元婴**: 金丹破碎，元婴出窍，可离体而战。
- **化神**: 元婴化神，领悟天地法则。
- **炼虚**: 炼虚合道，与天地大道相合。
- **合体**: 身体与道合一，实力通天。
- **大乘**: 渡劫前夕，领悟大乘佛法或大道真谛。
- **渡境**: 经历天劫，成功则飞升。
- **仙人**: 超脱凡尘，位列仙班。

## 待开发功能

### 1. 门派系统
- **门派任务**: 玩家可接受门派发布的任务，完成任务可获得门派贡献度、经验及其他奖励。任务类型多样，包括巡逻、采药、斩妖除魔等。
- **贡献度**: 玩家通过完成门派任务、上交资源等方式获得贡献度，贡献度可用于兑换门派资源、学习门派功法。
- **职位晋升**: 随着贡献度和实力的提升，玩家可在门派中获得更高的职位，解锁更多权限和福利。不同职位等级将对应不同的奖励，例如：
    - **外门弟子**: 可获得基础丹药、低阶法器、门派基础功法。
    - **内门弟子**: 可获得中阶丹药、中阶法宝、门派进阶功法，并可进入部分门派秘境。
    - **真传弟子**: 可获得高阶丹药、稀有法宝、门派核心功法，并可获得长老指点、参与门派核心事务。
    - **长老/护法**: 拥有门派管理权限，可获得丰厚资源供奉、专属洞府，并有机会接触门派镇派之宝。

### 2. 散修奇遇系统
- **秘境探索**: 散修玩家可自由探索各种秘境，秘境中蕴藏着珍稀的灵草、矿石、法宝和传承。
- **遗迹寻宝**: 发现上古遗迹，解开谜题，获得强大的功法或稀有宝物。
- **随机事件**: 散修在野外探索时，可能触发各种随机事件，如遭遇妖兽、发现洞府、遇到其他修仙者等。

### 3. 战斗系统
- **功法**: 玩家可学习和修炼各种功法，功法分为攻击、防御、辅助等类型，不同功法有不同的效果和成长路线。
- **法宝**: 玩家可收集、炼化和使用各种法宝，法宝能提供强大的战斗力或特殊效果。法宝有品阶之分，可通过炼化提升。
- **神通**: 玩家在达到特定境界或获得特殊机缘后，可领悟并施展强大的神通，神通往往具有毁天灭地的威能。

### 4. 资源管理
- **灵石**: 修仙界的通用货币，用于购买物品、修炼、交易等。可通过完成任务、出售资源、探索等方式获得。
- **丹药**: 用于恢复伤势、提升修为、突破境界等。玩家可学习炼丹术，自行炼制丹药。
- **材料**: 用于炼器、炼丹、布阵等。可通过采集、击杀妖兽、交易等方式获得。

### 5. 其他待开发功能
- 社交互动（与其他修仙者交流、结盟、仇杀）
- 渡劫系统
- 飞升机制

## 地域流派设定
修仙界广阔无垠，分为七大地域，每个地域都有其独特的修仙流派、资源和势力。玩家在探索过程中，将有机会接触到这些风格迥异的地域流派。

### 1. 东域 - 青木仙宗 (木系/治疗/辅助)
- **地域特色**: 灵气充沛，草木繁盛，盛产各种灵药仙草。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《青木引气诀》**: 青木仙宗的基础吐纳法门，帮助凡人感应木灵气，凝聚体内第一缕灵力，踏入炼气期。
        - **《小回春术》**: 初阶治疗法术，可用于治愈轻微伤势，恢复少量气血。
    - **筑基/金丹期 (进阶功法)**:
        - **《万木生发功》**: 中阶木系功法，修炼后可提升自身生命恢复速度，并小幅度增强木系法术威力。
        - **《藤蔓缠绕术》**: 释放坚韧藤蔓束缚敌人，使其短时间内无法移动并持续受到微弱木系伤害。
    - **元婴/化神期 (核心功法)**:
        - **《青帝长生诀》**: 高阶木系功法，可凝聚庞大生机之力，大幅提升治疗效果，甚至能续命吊魂。
        - **《木灵召唤术》**: 召唤木灵协助战斗或进行辅助，木灵具有治疗和缠绕能力。
- **丹药**
    - **凡人/炼气期**:
        - **养气丹**: 补充灵气，加速炼气期修为增长。
        - **小还丹**: 疗伤圣药，可迅速恢复凡人及炼气期修士的伤势。
    - **筑基/金丹期**:
        - **筑基丹**: 辅助筑基期修士稳固道基，提升突破金丹的成功率。
        - **生机丹**: 蕴含强大生机，可用于快速恢复中阶修士的生命和灵力。
    - **元婴/化神期**:
        - **蕴灵丹**: 大幅补充元婴期修士的灵力消耗，适合长时间战斗。
        - **清心丹**: 辅助化神期修士突破心魔，稳固道心。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **护心玉佩**: 凡品法器，可抵挡一次致命攻击，为炼气期修士提供基础防护。
        - **灵草锄**: 采药专用，能提升采集灵草的成功率和稀有度。
    - **筑基/金丹期 (中阶法宝)**:
        - **碧玉葫芦**: 内含生命精华，可瞬间恢复大量生命，并有几率解除致命伤。
        - **青木杖**: 蕴含生机之力，增强木系法术效果，并可召唤木灵协助战斗。
    - **元婴/化神期 (高阶法宝)**:
        - **万载青藤甲**: 以万年青藤炼制，防御力极强，并能自动修复，对木系法术有增幅。
        - **生命之源**: 稀世法宝，能源源不断提供生命能量，大幅延长寿元，辅助突破境界。
- **武器**:
    - **灵木剑**: 以千年灵木打造，轻盈锋利，附带木系毒素，可缓慢侵蚀敌人。
    - **翠竹弓**: 弓身由翠竹制成，射出的箭矢附带木系灵力，具有追踪和穿透效果。
- **NPC**:
    - **青木仙宗宗主 - 药尘子**: 一位慈眉善目的老者，精通炼丹之术，对门下弟子关爱有加。
    - **青木仙宗圣女 - 柳如烟**: 天资聪颖，擅长治疗法术，心地善良，常外出救助凡人。
    - **药园长老 - 谷神**: 负责门派药园管理，性格古板，但对灵药的培育有独到心得。

### 2. 南疆 - 焚天谷 (火系/爆发/群攻)
- **地域特色**: 火山密布，岩浆流淌，盛产火属性灵材和稀有矿石。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《火灵诀》**: 焚天谷的基础吐纳法门，引导凡人感应火灵气，使其具备初步控火能力。
        - **《炽焰术》**: 初阶火系攻击法术，可凝聚一团火焰攻击敌人，造成灼烧效果。
    - **筑基/金丹期 (进阶功法)**:
        - **《离火真经》**: 中阶火系功法，修炼后可大幅提升火系法术威力，并增强对火焰伤害的抗性。
        - **《火弹连珠》**: 连续发射多枚火弹，对目标造成持续打击。
    - **元婴/化神期 (核心功法)**:
        - **《焚天秘典》**: 高阶火系功法，修炼至大成可引动天地离火，焚烧万物，破坏力惊人。
        - **《炎龙咆哮》**: 召唤一条火焰巨龙冲击敌人，造成巨额火系范围伤害。
- **丹药**
    - **凡人/炼气期**:
        - **火灵丹**: 蕴含火灵之力，提升炼气期火系功法修炼速度。
        - **清热丹**: 解毒，并缓解初阶修士体内燥热，辅助修炼火系功法。
    - **筑基/金丹期**:
        - **爆炎丹**: 短时间内大幅提升火系法术的爆发力。
        - **淬火丹**: 淬炼体魄，增强筑基、金丹期修士对火焰伤害的抗性。
    - **元婴/化神期**:
        - **炽阳丹**: 蕴含纯阳之力，可帮助元婴期修士突破瓶颈，增强阳刚之气。
        - **涅槃丹**: 稀世丹药，可在重伤濒死之际激发体内潜力，短暂恢复战力。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **控火珠**: 辅助炼气期修士更好地操控火焰，提升法术精准度。
        - **烈火符**: 简单符箓，激发后可释放一道火焰攻击，对付低阶妖兽有效。
    - **筑基/金丹期 (中阶法宝)**:
        - **赤焰珠**: 蕴含纯粹火灵，可大幅提升火系法术威力，并能抵挡部分火焰伤害。
        - **离火扇**: 轻轻一扇，便可引动天地离火，焚烧万物，对妖魔有奇效。
    - **元婴/化神期 (高阶法宝)**:
        - **朱雀焚天鼎**: 焚天谷镇派法宝之一，可凝聚天地火灵，炼化万物，亦可作为攻击法宝释放焚天之火。
        - **炎帝之怒**: 蕴含远古炎帝一丝神识的法宝，激活后可召唤炎帝虚影，释放毁灭性火系攻击。
- **武器**:
    - **烈焰刀**: 刀身常年被地火淬炼，刀锋附带灼热火焰，攻击时可点燃敌人。
    - **火云枪**: 枪尖吞吐火舌，舞动间如火云翻腾，招式大开大合，势不可挡。
- **NPC**:
    - **焚天谷谷主 - 炎无极**: 性格火爆，实力强悍，以霸道手段镇守南疆。
    - **炼器大师 - 欧冶子**: 醉心于炼器之道，常年与地火为伴，能打造出顶尖火系法宝。
    - **火灵儿**: 谷中诞生的火系精灵，活泼好动，对谷主忠心耿耿。

### 3. 西漠 - 金刚寺 (土系/防御/体修)
- **地域特色**: 黄沙漫天，戈壁荒凉，灵气稀薄但蕴含厚重土属性。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《金刚锻体诀》**: 金刚寺的基础炼体法门，强健肉身，引导土灵气入体，为后续修炼打下基础。
        - **《磐石术》**: 初阶防御法术，可在体表形成一层薄薄的土灵气护甲，抵御轻微攻击。
    - **筑基/金丹期 (进阶功法)**:
        - **《厚土真经》**: 中阶土系功法，修炼后可大幅提升肉身防御力，并增强土系法术的威能。
        - **《地动术》**: 震动大地，对周围敌人造成轻微土系伤害并使其短暂眩晕。
    - **元婴/化神期 (核心功法)**:
        - **《金刚不坏体神功》**: 金刚寺镇派炼体神功，修炼至大成可使肉身坚不可摧，免疫大部分物理伤害，甚至能抵御部分法术攻击。
        - **《不动明王印》**: 结出庄严法印，大幅提升自身防御和抗性，同时对周围敌人造成土系震慑。
- **丹药**
    - **凡人/炼气期**:
        - **炼体丹**: 辅助炼气期修士淬炼肉身，提升体质。
        - **固本丹**: 稳固凡人及炼气期修士的根基，增强抗击打能力。
    - **筑基/金丹期**:
        - **铜皮丹**: 提升筑基期修士肉身强度，使其防御力媲美低阶法宝。
        - **厚土丹**: 增强金丹期修士对土系灵气的亲和力，提升土系功法修炼速度。
    - **元婴/化神期**:
        - **金刚丹**: 大幅增强元婴期修士的肉身强度和防御，是体修突破境界的必备丹药。
        - **镇岳丹**: 辅助化神期修士领悟厚土法则，稳固道基，增强对天地法则的抗性。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **铁布衫**: 凡品法衣，可提供基础物理防御。
        - **沙袋**: 辅助炼体，可加速肉身强度的提升。
    - **筑基/金丹期 (中阶法宝)**:
        - **玄武甲**: 以玄武之壳炼制，防御力惊人，可反弹部分伤害。
        - **镇魔杵**: 沉重无比，蕴含镇压之力，可破除邪法，对魔修有克制作用。
    - **元婴/化神期 (高阶法宝)**:
        - **不灭金身舍利**: 金刚寺高僧圆寂后所留，蕴含佛门金身之力，佩戴者可获得强大的防御和恢复能力。
        - **九天息壤盾**: 以传说中的九天息壤炼制，可随意变化大小，防御力惊世骇俗，能抵挡仙人一击。
- **武器**:
    - **降魔棍**: 寺中高僧所用，棍身刻满佛门符文，挥舞间佛光普照，降妖伏魔。

- **NPC**:
    - **金刚寺方丈 - 慧能大师**: 佛法高深，修为通天，以慈悲之心普度众生。
    - **武僧首座 - 悟净**: 寺中武力最强者，擅长近身搏斗，肉身强悍。

### 4. 北原 - 冰雪宫 (水/冰系/控制)
- **地域特色**: 冰天雪地，寒风凛冽，盛产冰属性灵材和寒铁。

        - **《冰灵诀》**: 冰雪宫的基础吐纳法门，帮助凡人感应冰灵气，凝聚寒气。
        - **《冰锥术》**: 初阶冰系攻击法术，可凝聚冰锥攻击敌人，造成冻伤。
    - **筑基/金丹期 (进阶功法)**:
        - **《玄冰心法》**: 中阶冰系功法，修炼后可增强冰系法术的控制效果和伤害，并提升对寒冷的抵抗力。
        - **《寒冰护体》**: 召唤一层冰甲附着自身，提升防御并对近战敌人造成冰冻效果。
    - **元婴/化神期 (核心功法)**:
        - **《冰魄寒光诀》**: 高阶冰系功法，修炼至大成可操控极致寒意，冰封万里，甚至能冻结时间，具有强大控制和爆发。
        - **《冰封千里》**: 释放极寒之气，将大范围敌人冰冻，使其无法行动。
- **丹药**
    - **凡人/炼气期**:
        - **凝露丹**: 补充水灵气，加速炼气期水/冰系功法修炼。
        - **暖身丹**: 抵抗北原严寒，辅助初阶修士在恶劣环境中修炼。
    - **筑基/金丹期**:
        - **玄冰丹**: 提升筑基期修士对冰系法术的领悟，增强控制效果。
        - **寒晶丹**: 蕴含精纯寒力，辅助金丹期修士提升修为，并增强抗寒能力。
    - **元婴/化神期**:
        - **冰魄丹**: 大幅提升元婴期修士的冰系法术威力及冰冻控制效果。

    - **凡人/炼气期 (低阶法器)**:
        - **御寒符**: 激发后可抵御一定寒气，保护炼气期修士不受冻伤。
        - **冰晶手镯**: 凡品法器，可凝聚微弱冰晶攻击敌人，具有减速效果。
    - **筑基/金丹期 (中阶法宝)**:
        - **玄冰镜**: 可反射法术攻击，并能释放寒气冰冻敌人。
        - **雪魄珠**: 蕴含极致寒意，可用于修炼冰系功法，或在战斗中释放冰霜之力。
    - **元婴/化神期 (高阶法宝)**:
        - **冰魄神剑**: 冰雪宫镇派法宝之一，剑身由万年寒冰凝结，拥有冰封天地之能。
        - **九天玄冰塔**: 释放强大冰系结界，镇压敌人，并能吸收天地寒气为己所用。
- **武器**:
    - **冰魄剑**: 剑身由万年寒冰凝结，锋利无比，攻击附带冰冻减速效果。

    - **冰雪宫宫主 - 冰心仙子**: 冷若冰霜，不苟言笑，但内心善良，守护北原生灵。

    - **寒冰老人**: 隐居在极寒之地，精通冰系阵法和符箓。

### 5. 中州 - 天剑阁 (金系/剑修/单体)
- **地域特色**: 灵气最为浓郁，资源丰富，是人族修仙文明的中心。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《引剑诀》**: 天剑阁的基础引气法门，引导凡人感应金灵气，初步凝聚剑气。
        - **《基础剑法》**: 天剑阁入门剑法，招式简单却蕴含剑道至理，修炼后可掌握基础剑术。
    - **筑基/金丹期 (进阶功法)**:
        - **《庚金剑诀》**: 中阶金系功法，修炼后可大幅提升剑气锋锐度，增强剑法威力。
        - **《御剑术》**: 掌握御剑飞行之法，可在空中自由穿梭，并操控飞剑进行攻击。
    - **元婴/化神期 (核心功法)**:
        - **《万剑归宗》**: 天剑阁镇派剑诀之一，修炼至大成可召唤无数剑影攻击敌人，造成毁灭性金系范围伤害。
        - **《一剑破万法》**: 凝聚全身剑意，对单个目标造成无视防御的致命一击，是极致单体爆发剑招。
- **丹药**
    - **凡人/炼气期**:
        - **炼气丹**: 补充灵气，加速炼气期修为增长。
        - **金锋丹**: 辅助炼气期剑修淬炼剑气，使其更加锋利。
    - **筑基/金丹期**:
        - **剑心丹**: 提升筑基期剑修的剑意领悟，增强剑法威力。
        - **凝剑丹**: 辅助金丹期剑修凝聚金丹剑胎，为元婴期打下基础。
    - **元婴/化神期**:
        - **破障丹**: 帮助元婴期剑修突破瓶颈，领悟更高层次剑道。
        - **天剑丹**: 稀世丹药，可短暂提升化神期剑修的剑道境界，施展出超越自身极限的剑招。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **精铁飞剑**: 凡品法器，可初步御使，进行简单攻击。
        - **磨剑石**: 辅助剑修磨砺剑器，提升其锋利度。
    - **筑基/金丹期 (中阶法宝)**:
        - **斩仙飞刀**: 传说中的至宝，一旦祭出，可斩仙人，威力无穷 (此为传说，此处指仿制品或残缺品)。
        - **七星剑匣**: 内含七把飞剑，可同时御使，组成剑阵攻击敌人。
    - **元婴/化神期 (高阶法宝)**:
        - **紫霄剑**: 天剑阁镇派之宝，剑气纵横，可斩断虚空，拥有自我修复能力。
        - **诛仙剑阵图**: 稀有法宝，内含诛仙剑阵，一旦布下，可困杀仙人。
- **武器**:
    - **紫霄剑**: 天剑阁镇派之宝，剑气纵横，可斩断虚空。
    - **流光剑**: 轻盈迅捷，剑法飘逸，适合快速连击。
- **NPC**:
    - **天剑阁阁主 - 剑无尘**: 剑道巅峰强者，一剑可开山断海，为人正直。
    - **铸剑大师 - 欧阳锋**: 擅长铸造各种神兵利器，对剑的理解无人能及。
    - **剑痴 - 独孤求败**: 终日沉迷剑道，四处挑战强者，只为追求剑道极致。

### 6. 东海 - 碧波岛 (水系/召唤/毒系)
- **地域特色**: 岛屿众多，海洋广阔，盛产水属性灵材和海洋妖兽。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《碧水诀》**: 碧波岛的基础吐纳法门，帮助凡人感应水灵气，增强对水元素的亲和力。
        - **《水箭术》**: 初阶水系攻击法术，凝聚水箭攻击敌人。
    - **筑基/金丹期 (进阶功法)**:
        - **《碧海潮生功》**: 中阶水系功法，修炼后可操控小范围水流，并增强水系法术的威能。
        - **《毒雾弥漫》**: 释放剧毒迷雾，使范围内敌人持续中毒并降低属性。
    - **元婴/化神期 (核心功法)**:
        - **《沧海龙吟诀》**: 高阶水系功法，修炼至大成可引动四海之力，召唤滔天巨浪，具有强大的范围攻击和控制能力。
        - **《海妖召唤术》**: 召唤强大的海妖协助战斗，海妖拥有各自的特殊能力，可随修为提升召唤更强大的海妖。
- **丹药**
    - **凡人/炼气期**:
        - **清灵丹**: 补充灵气，加速炼气期修为增长。
        - **避水丹**: 短时间内可在水中自由呼吸，辅助初阶修士水下探索。
    - **筑基/金丹期**:
        - **解毒丹**: 解除各种中低阶毒素，是外出探索必备。
        - **化形丹**: 辅助部分水族妖兽化形，对人族修士无效。
    - **元婴/化神期**:
        - **龙元丹**: 蕴含龙族精华，可大幅提升元婴期修士的水系法术威力，并增强肉身强度。
        - **通灵丹**: 辅助化神期修士与海妖建立更深层次的联系，提升召唤海妖的成功率和强度。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **鱼鳞甲**: 凡品法衣，由坚韧鱼鳞制成，提供水下防护。
        - **水灵珠**: 凡品法器，可凝聚水球攻击敌人，并能在水中提供微弱光芒。
    - **筑基/金丹期 (中阶法宝)**:
        - **定海神珠**: 传说中可定海平波的至宝，可操控水流，制造漩涡。
        - **避水珠**: 佩戴后可在水中自由呼吸，并大幅提升水下战斗能力。
    - **元婴/化神期 (高阶法宝)**:
        - **龙王三叉戟**: 碧波岛镇派法宝之一，蕴含真龙之力，攻击附带水系冲击，可击飞敌人，并能操控局部海域。
        - **潮汐宝珠**: 能引动海洋潮汐之力，制造范围性巨浪攻击，并能治愈水系修士伤势。
- **武器**:
    - **三叉戟**: 海族强者所用，攻击附带水系冲击，可击飞敌人。
    - **珊瑚杖**: 以深海珊瑚制成，增强水系法术效果，并可召唤水元素。
- **NPC**:
    - **碧波岛岛主 - 龙女敖心**: 东海龙王之女，性格温婉，但实力深不可测。
    - **海妖统领 - 蛟魔王**: 统领东海群妖，实力强大，对人族抱有敌意。
    - **渔夫 - 老张**: 世代居住在碧波岛，对东海的各种传说和秘闻了如指掌。

### 7. 幽冥 - 鬼王殿 (暗系/诅咒/亡灵)
- **地域特色**: 阴气森森，鬼气弥漫，是亡灵和鬼修的聚集地。
- **功法**
    - **凡人/炼气期 (基础功法)**:
        - **《冥灵诀》**: 鬼王殿的基础引气法门，引导凡人感应阴气，初步掌控亡魂之力。
        - **《幽魂术》**: 初阶暗系攻击法术，可召唤微弱幽魂攻击敌人，造成精神影响。
    - **筑基/金丹期 (进阶功法)**:
        - **《万鬼噬魂功》**: 中阶暗系功法，修炼后可召唤更多恶鬼攻击敌人，吸取其生命和灵魂，增强自身。
        - **《死亡诅咒》**: 对单个目标施加恶毒诅咒，使其持续受到伤害并降低部分属性。
    - **元婴/化神期 (核心功法)**:
        - **《九幽冥典》**: 鬼王殿镇派功法之一，修炼至大成可掌控生死轮回，召唤强大的亡灵大军，释放毁天灭地的暗系神通。
        - **《亡灵君主》**: 召唤强大的亡灵生物（如高阶骷髅、僵尸王、幽魂领主）协助战斗，这些亡灵拥有各自的特殊能力。
- **丹药**
    - **凡人/炼气期**:
        - **聚阴丹**: 补充阴气，加速炼气期鬼修功法修炼。
        - **敛息丹**: 隐藏凡人及炼气期修士的生人气息，方便在幽冥地界行动。
    - **筑基/金丹期**:
        - **魂力丹**: 提升筑基期鬼修的灵魂强度，增强对亡魂的控制力。
        - **蚀骨丹**: 蕴含强大腐蚀之力，可用于炼制毒物或直接攻击敌人。
    - **元婴/化神期**:
        - **幽冥丹**: 大幅提升元婴期鬼修的暗系法术威力，并增强对亡灵的控制力。
        - **往生丹**: 稀世丹药，可帮助化神期鬼修突破轮回桎梏，领悟生死法则。
- **法宝**
    - **凡人/炼气期 (低阶法器)**:
        - **骨哨**: 凡品法器，吹响后可召唤一两只低阶骷髅或幽魂辅助战斗。
        - **腐毒符**: 简单符箓，激发后可释放一团腐蚀毒气，对付生灵有效。
    - **筑基/金丹期 (中阶法宝)**:
        - **招魂幡**: 可召唤亡魂为己所用，并能吸收死者怨气增强自身。
        - **幽冥珠**: 蕴含幽冥之力，可用于修炼暗系功法，或在战斗中释放死亡气息。
    - **元婴/化神期 (高阶法宝)**:
        - **鬼王令**: 鬼王殿殿主信物，持有者可调动幽冥鬼兵，号令群鬼，是权力的象征。
        - **生死簿碎片**: 传说中生死簿的碎片，蕴含一丝生死法则，可短暂操控生灵寿元，威力惊人。
- **武器**:
    - **噬魂镰**: 镰刀上附着亡魂怨气，攻击可吸取敌人生命，并有几率造成恐惧。
    - **白骨法杖**: 以万年白骨炼制，增强暗系法术效果，并可操控亡灵。
- **NPC**:
    - **鬼王殿殿主 - 幽冥鬼王**: 幽冥界至强者，掌控生死轮回，性格阴沉。
    - **判官 - 崔钰**: 负责审判亡魂，铁面无私，但有时也会网开一面。
    - **孟婆**: 守候在奈何桥边，为亡魂提供孟婆汤，使其忘却前尘。
